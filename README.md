# LSTM-Path-Predictor
This project implements a Recurrent Neural Network (RNN) to predict the trajectory of an individual in a crowded room based on various environmental and positional data. The model is trained using sequential data to forecast the future position of a person as they navigate through obstacles and move toward a goal.

LSTM Architecture: Utilizes an LSTM to model sequential data, specifically focusing on the movement of individuals.

Positional Data: The model is trained on features such as position, speed, and environmental factors (e.g., obstacles, goals).

Crowded Room: The dataset simulates crowded environments with dynamic variables that influence the person’s movement.

Prediction Output: Predicts the next position of the individual based on the current state and previous movement.
